STYLE PROCESS

Possible Designs

Defining Visual Aesthetic

My website will showcase a sleek, modern and immersive look using blacks or rich tones for the background. There will be vibrant accent colours which contrast against the dark background. This contrast of colours, will enhance readability, ensuring that the text and UI remains clear. This use of colour palate showcases creativity, as the dark background will allow for the colourful visuals to stand out. This will allow for my portfolio work to stand out.

Reference material

https://www.stevenmengin.com/

I like the dark thematic feel of Steven Mengin’s websites. The background of the website is black, and their pops of colour in the images. I will incorporate this into my website, but I would wat to add more colour, in text and in icons. In this website, you have cursor which changes into an arrow when you hover over images, allowing you to switch between images. I like this idea; I am not sure if I will implement it but will definitely consider adding more creative elements like this. At the footer of the website, if you hover over certain texts, the background image changes. This is a creative idea, which I will be adding. This is something that will align with my goal of creativity.

https://thieb.co/

in this website, it continues with the dark aesthetic and pops of colour which I want to go into. As you are scrolling the website, the text and images fade onto the screen. This is something I would like to add to my website, it is creative and allows for readability. When the text fades in, it stays static, allowing for users to read the information.
